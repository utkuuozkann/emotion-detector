# Emotion Detector

This project is a sentiment analysis application built with Python and Flask.

## Features
- Detects emotions from text input
- Returns anger, disgust, fear, joy, and sadness
- Identifies dominant emotion
- Flask API support
- Unit testing included

## Project Structure
- EmotionDetection/
- server.py
- test_emotion_detection.py

## Technologies
- Python
- Flask
- Watson NLP
- unittest
- pylint
